This repository contains all supplementary materials associated with the master’s thesis “AI-Driven Interview-Based Process Discovery: A Conversational Agent for Eliciting Process Knowledge and Generating Process Models.” 
The repository provides the full artifact implementation and supporting research materials to ensure transparency, reproducibility, and extensibility of the presented work. Specifically, it includes (1) the source code of the developed prototype, including agent configurations and transformation logic, (2) evaluation data sets, ground-truth process models, generated process models, and computed evaluation results, and (3) the complete data basis for the systematic literature review, including screening results. The materials allow researchers and practitioners to reproduce the evaluation, analyse the artifact design, and build upon the proposed LLM-driven interview-based process discovery framework.

To implement the artifact follow the ChatKit quickstart instructions and use the following workflow ID: wf_692970cc7c24819097469282d69026020a47eaf2fdc30913. For this a valid OpenAI API key is needed. After the successfull implementation the chat interface with the Interview Agent can be opened and used. The final output in the chat, which comes from the Transformer Agent, needs to be stored in a text file and given to the BPMN XML Generator component. 
This last component can also be implemented locally by downloading the code and setting it up for example in VS Code or Jupyter Notebook. For a successful execution of the code also the docker engines need to run.
